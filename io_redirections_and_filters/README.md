# IO Redirections and Filters

This directory/file contains exercises related to input/output redirections and filters in Bash.

## Purpose

- Learn how to redirect standard output and standard error.
- Use pipes to connect commands.
- Filter text using commands like grep, sort, cut, uniq, and wc.
- Combine redirections and filters in Bash scripts.

## Usage

- Run commands or scripts from the terminal.
- Redirect output to files using > or >>.
- Redirect errors with 2> or 2>&1.
- Filter output using pipes (|) and commands like grep.
